🌐 代理设置

为目标网站配置统一的代理地址。所有数据源（DailyView 等）均共享此代理设置。

代理配置

📖 Windows 查看代理端口的方法：
1. 打开代理软件（Clash / V2RayN / Shadowsocks 等）主界面
2. 查看软件界面显示的 HTTP/SOCKS5 端口号
3. 或打开 Windows 设置 → 网络和 Internet → 代理 查看地址和端口
4. 常见默认端口：
   Clash http://127.0.0.1:7890  |  V2Ray socks5://127.0.0.1:10808
   Shadowsocks socks5://127.0.0.1:1080  |  Clash Verge http://127.0.0.1:7897
