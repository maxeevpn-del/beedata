============================================
  MFData - DailyView 周榜数据抓取工具
  用户手册
============================================

一、软件简介
----------
本工具用于抓取 DailyView（网络温度计）网站的话题周榜数据，并导出为 Excel 文件。
支持热门话题分类（台剧、电影、综艺、3C 家电等60+分类），可按日/周/月查看排行榜。

二、安装步骤（首次使用）
----------------------
1. 确保电脑已安装 Node.js
   - 如果未安装，请访问 https://nodejs.org 下载安装 LTS 版本
   - 安装时一路点"下一步"即可

2. 双击项目文件夹中的 install.bat
   - 自动安装程序依赖
   - 自动在桌面创建 "MFData" 快捷方式

3. 安装完成！双击桌面的 "MFData" 图标即可启动。

三、启动方式
----------
方式一（推荐）：双击桌面 "MFData" 快捷方式
方式二：双击项目文件夹中的 "启动工具.bat"

启动后浏览器会自动打开，访问地址为：http://localhost:3456

四、日常使用指南
--------------

【第1步：配置代理】
由于 DailyView 网站需要翻墙访问，必须先配置代理：
1. 点击左侧菜单「全局设置」
2. 在"代理地址"输入框中填入你的代理地址，例如：
      http://127.0.0.1:7890    （Clash 默认端口）
      socks5://127.0.0.1:1080  （其他代理）
3. 选择代理类型（HTTP 或 SOCKS5）
4. 点击「测试连接」按钮，确认能连接成功
5. 点击「保存并应用」保存设置

如何查看 Windows 代理端口？
- 打开你的代理软件主界面，查看显示的端口号
- 或打开 Windows 设置 → 网络和 Internet → 代理，查看代理信息

【第2步：抓取数据】
1. 点击左侧菜单「DailyView 周榜」
2. 选择话题分类（默认：47 - 娱乐/台剧）
3. 选择时间范围（今日/本周/本月）
4. 点击「抓取数据」按钮
5. 等待进度条完成，数据表格自动展示

【第3步：导出 Excel】
1. 数据展示后，点击「📥 导出 Excel」按钮
2. Excel 文件自动保存到项目文件夹
3. 浏览器会自动下载该文件
4. 历史记录可在「抓取记录」页面查看和重新下载

【补充功能】
- 分页选择器：表格右上角可选择每页显示 10/20/30/50 条
- 打印预览：点击「🖨 打印预览」可打印当前数据
- 抓取记录：左侧「📋 抓取记录」可查看所有历史操作记录

五、常见问题
----------
Q：双击启动工具闪退？
A：请确认已安装 Node.js。如未安装，先运行 install.bat。

Q：提示端口已被占用？
A：说明程序已在运行，直接打开浏览器访问 http://localhost:3456 即可。

Q：抓取数据失败，提示连接超时？
A：检查代理配置是否正确。进入「全局设置」→「测试连接」验证代理是否可用。

Q：导出 Excel 后找不到文件？
A：Excel 文件保存在项目文件夹中，文件名格式为：
   dailyview_话题47_周榜_20260117_0930.xlsx
